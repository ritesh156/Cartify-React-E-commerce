# Cartify – E-commerce Website

Cartify is a basic e-commerce web application built using React and Node.js. It demonstrates a simple shopping interface along with user authentication using JWT.

## Features

* User login with JWT authentication
* Protected routes for authorized users
* Simple product display UI
* Component-based React structure
* Basic client-server interaction

## Tech Stack

* React.js
* Node.js
* Express.js
* JSON Web Token (JWT)

## Project Structure

* `client/` → React frontend
* `server/` → Node.js backend

## How to Run

1. Clone the repository
2. Install dependencies in both folders:

   * `cd client && npm install`
   * `cd server && npm install`
3. Start backend:

   * `node server.js`
4. Start frontend:

   * `npm start`

## Note

This is a basic implementation. It can be extended by adding database integration, cart functionality, and payment features.
